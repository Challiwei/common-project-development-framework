## 通用说明

### 请求头

| Header | 值 | 是否必须 | 说明 |
|--------|------|----------|------|
| Content-Type | application/json | 是 | 请求格式 |
| Authorization | Bearer {token} | 是（除登录外） | 认证令牌 |

### 通用响应格式

**成功**：
```json
{
  "code": "SUCCESS",
  "data": {},
  "message": "操作成功"
}
```

**失败**：
```json
{
  "code": "ERROR_CODE",
  "message": "错误描述",
  "details": {}
}
```

### 分页参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| page | int | 1 | 页码 |
| page_size | int | 20 | 每页数量（最大100） |

### 分页响应

```json
{
  "code": "SUCCESS",
  "data": {
    "items": [],
    "total": 0,
    "page": 1,
    "page_size": 20
  }
}
```
